# uNews

**uNews** — Telegram-бот и система публикации новостей, патчноутов и отчётов разработки по u-проектам.

Проект предназначен для автоматической или полуавтоматической публикации обновлений из GitHub-репозиториев в Telegram-канал.

## Назначение

**uNews** помогает вести единый Telegram-канал с новостями разработки:

- публиковать патчноуты;
- публиковать отчёты о разработке;
- отправлять новости проектов;
- прикреплять изображения к каждому посту;
- хранить историю обновлений в GitHub;
- использовать GitHub Actions для автоматической публикации.

## Основные проекты

- uSugar
- uDream
- uChurch
- GOART
- Time Rift
- другие проекты автора

## Как работает система

1. В папку `news/` добавляется файл патчноута.
2. Рядом добавляется изображение с таким же именем.
3. GitHub Actions запускает скрипт публикации.
4. Скрипт отправляет пост в Telegram-канал.
5. Telegram-канал получает полноценный патчноут с картинкой.
